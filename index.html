查询积分

请求地址

https://www.abc0515.com/wxappapi/api/query/GetUserPoint

请求方式

POST

请求参数

字段 类型 说明
storeId int 商户id

请求格式

https://www.abc0515.com/wxappapi/api/query/GetUserPoint?opneId=xxxx&sign=xxx&storeId=123

###返回参数

字段 类型 归属 说明
result object 查询结果
userId int result 用户id
userName string result 用户名字
storeId int result 商户id
storeName string result 商户名
point int result 积分
detail list result 积分明细
point int detail 积分
cost int detail 花费
createtime datetimeint detail 创建时间

获取会员和商户的关系

请求地址

https://www.abc0515.com/wxappapi/api/query/GetReleation

请求方式

POST

请求参数

字段 类型 说明
storeId int 商户id，不传自动获取用户关注的商户
isStore int 查询商户对应会员是必传

请求格式

url形式

https://www.abc0515.com/wxappapi/api/query/GetReleation?opneId=xxxx&sign=xxx

json格式

返回参数

字段 类型 归属 说明
result object 查询结果
userId int result 用户id
userName string result 用户名字
openId string result 返回结果
storeId int result 商户id
storeName string result 商户名

接口请求说明

请求方式

POST

请求格式

{url}?opneId={openId}&sign={sign}

说明

字段 说明
url 请求地址
openId 用户openId
sign 签名

签名说明

signKey：1aec44863bc84556b7903df11478d5d9

签名格式：{signKey}&{openId}，例如：1aec44863bc84556b7903df11478d5d9&isijkxkaoapxla
然后进行MD5
