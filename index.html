网络编程简介

协议层

TCP

  1. 有连接
  2. 面向数据流
  3. 可靠的点对点协议
  • 优点：可靠
  • 缺点：需要额外的宽带

UDP

  1. 无连接
  2. 面向数据包，传输大数据包
  3. 无序的

EF各种模式简介

DBFirst

  1. 建立类，类名与表明相同，属性与相应数据库字段对应，名称要保持一致
  2. 建立 DBContext
    • 继承DbContext
    • 创建数据库映射，其中Test为数据库中的表，DbSet可以看作是数据集合
      public virtual DbSet<Test> Test { get; set; }
    • 重写OnModelCreating
      protected override void OnModelCreating(DbModelBuilder modelBuilder)
{
  //移除数据库自动映射
  modelBuilder.Conventions.Remove<PluralizingTableNameConvention>();
  base.OnModelCreating(modelBuilder);
}
