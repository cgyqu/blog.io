获取会员和商户的关系

获取会员和商户的关系

请求地址

https://www.abc0515.com/wxappapi/api/query/GetReleation

请求方式

POST

请求参数

字段 类型 说明
storeId int 商户id，不传自动获取用户关注的商户
isStore int 查询商户对应会员是必传

请求格式

https://www.abc0515.com/wxappapi/api/query/GetReleation?opneId=xxxx&sign=xxx

返回参数

字段 类型 归属 说明
result object 查询结果
userId int result 用户id
userName string result 用户名字
openId string result 返回结果
storeId int result 商户id
storeName string result 商户名
